# Lux-Cafe
Website institucional elegante e totalmente responsivo (Mobile-First) desenvolvido para a cafeteria de alta qualidade Lux Cafe. O projeto utiliza HTML5, CSS3 e JavaScript puro para apresentar de forma clara e sofisticada o cardápio, horários e a filosofia da marca, garantindo uma experiência de usuário limpa e profissional em qualquer dispositivo.
